Introduction to PHP

PHP, which stands for "Hypertext Preprocessor," is a widely used server-side scripting language that is particularly well-suited for web development. Originally created by Danish-Canadian programmer Rasmus Lerdorf in 1994, PHP has since evolved into a powerful and versatile tool for building dynamic websites and web applications.

One of PHP's key strengths is its seamless integration with HTML, making it easy for developers to embed PHP code directly within HTML pages. This allows for the dynamic generation of content, such as retrieving data from a database, processing user input, or performing calculations, and then displaying the results on the webpage.

Installing PHP

To start using PHP, you have a few options:

1. Use a Web Host with PHP Support:

  • If your web host supports PHP, you can simply create PHP files, place them in your web directory, and the server will automatically parse them for you.
  • Most web hosts offer PHP support since PHP is free to use.

2. Set Up PHP on Your Own PC:

  • If your server does not support PHP, you can set it up on your own PC by following these steps:
  • Install a web server: You'll need to install software like Apache or Nginx to serve your PHP files.
  • Install PHP: Download and install PHP on your PC. The official PHP website provides installation instructions.
  • Install a database: If your application requires a database, such as MySQL, you'll need to install and configure it on your PC as well.

3. Use an Online PHP Compiler/Editor:

  • Websites like W3Schools offer online PHP compilers/editors where you can write PHP code directly in your browser.
  • These platforms allow you to edit PHP code and instantly view the result in your browser without needing to set up a server environment on your PC.
PHP Syntax

Basic PHP Syntax:

A PHP script starts with <?php and ends with ?>.

The default file extension for PHP files is .php.

A PHP file normally contains HTML tags and some PHP scripting code.

Example of a Simple PHP File:

A simple PHP file with both HTML code and PHP code:

<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>

Note: PHP statements end with a semicolon (;).

PHP Case Sensitivity:

In PHP, keywords (e.g., if, else, while, echo, etc.), classes, functions, and user-defined functions are not case-sensitive.

However, all variable names are case-sensitive.

Example demonstrating case sensitivity:

<!DOCTYPE html>
<html>
<body>

<?php
$color = "red";
echo "My car is " . $color . "<br>";
echo "My house is " . $COLOR . "<br>";
echo "My boat is " . $coLOR . "<br>";
?>

</body>
</html>
PHP Magic Constants
Constant Description Example
__CLASS__ If used inside a class, the class name is returned. echo __CLASS__;
__DIR__ The directory of the file. echo __DIR__;
__FILE__ The file name including the full path. echo __FILE__;
__FUNCTION__ If inside a function, the function name is returned. echo __FUNCTION__;
__LINE__ The current line number. echo __LINE__;
__METHOD__ If used inside a function that belongs to a class, both class and function name is returned. echo __METHOD__;
__NAMESPACE__ If used inside a namespace, the name of the namespace is returned. echo __NAMESPACE__;
__TRAIT__ If used inside a trait, the trait name is returned. echo __TRAIT__;
ClassName::class Returns the name of the specified class and the name of the namespace, if any. echo ClassName::class;
PHP If...Else

PHP Conditional Statements

Very often when you write code, you want to perform different actions for different conditions. You can use conditional statements in your code to do this.

In PHP, we have the following conditional statements:

  • if statement: executes some code if one condition is true.
  • if...else statement: executes some code if a condition is true and another code if that condition is false.
  • if...elseif...else statement: executes different codes for more than two conditions.
  • switch statement: selects one of many blocks of code to be executed.

Below are the basic syntaxes for these conditional statements:

if Statement:


if (condition) {
  // code to be executed if condition is true;
}

if...else Statement:


if (condition) {
  // code to be executed if condition is true;
} else {
  // code to be executed if condition is false;
}

if...elseif...else Statement:


if (condition1) {
  // code to be executed if condition1 is true;
} elseif (condition2) {
  // code to be executed if condition2 is true;
} else {
  // code to be executed if all conditions are false;
}

switch Statement:


switch (variable) {
  case value1:
    // code to be executed if variable is equal to value1;
    break;
  case value2:
    // code to be executed if variable is equal to value2;
    break;
  ...
  default:
    // code to be executed if variable is different from all the above values;
}
PHP Loops

PHP Loops

Oftentimes when you write code, you want the same block of code to run over and over again a certain number of times. So, instead of adding several almost equal code-lines in a script, we can use loops.

Loops are used to execute the same block of code again and again, as long as a certain condition is true.

In PHP, we have the following loop types:

  • while: loops through a block of code as long as the specified condition is true.
  • do...while: loops through a block of code once and then repeats the loop as long as the specified condition is true.
  • for: loops through a block of code a specified number of times.
  • foreach: loops through a block of code for each element in an array.

Below are the basic syntaxes for these loop types:

while Loop:


while (condition) {
  // code to be executed as long as the condition is true;
}

do...while Loop:


do {
  // code to be executed at least once, and then repeats the loop as long as the condition is true;
} while (condition);

for Loop:


for (initialization; condition; increment/decrement) {
  // code to be executed for each iteration of the loop;
}

foreach Loop:


foreach ($array as $value) {
  // code to be executed for each element in the array;
}
PHP Object-Oriented Programming (OOP)

PHP Object-Oriented Programming (OOP)

From PHP 5 onwards, you can also write PHP code in an object-oriented style.

What is OOP?

OOP stands for Object-Oriented Programming.

Procedural programming is about writing procedures or functions that perform operations on the data, while object-oriented programming is about creating objects that contain both data and functions.

Object-oriented programming has several advantages over procedural programming:

  • OOP is faster and easier to execute.
  • OOP provides a clear structure for the programs.
  • OOP helps to keep the PHP code DRY "Don't Repeat Yourself", and makes the code easier to maintain, modify, and debug.
  • OOP makes it possible to create full reusable applications with less code and shorter development time.

Tip: The "Don't Repeat Yourself" (DRY) principle is about reducing the repetition of code. You should extract out the codes that are common for the application, and place them at a single place and reuse them instead of repeating it.

What are Classes and Objects?

Classes and objects are the two main aspects of object-oriented programming.

PHP Object-Oriented Programming (OOP) Examples

Example 1: Class Declaration


class Car {
  // Properties
  public $brand;
  public $model;
  
  // Methods
  public function drive() {
    return "The {$this->brand} {$this->model} is driving.";
  }
}

Example 2: Object Instantiation


// Create an instance of the Car class
$myCar = new Car();

// Set object properties
$myCar->brand = "Toyota";
$myCar->model = "Camry";

Example 3: Accessing Properties and Methods


// Accessing object properties
echo "My car is a {$myCar->brand} {$myCar->model}.
";

// Accessing object methods
echo $myCar->drive();
